AsciiControl

AsciiControl enum Introduction

An escaped code that represents a single ASCII control character.

While not escape codes in the traditional sense, these characters are considered escape codes in the context of ANSI escape codes, as they are used to control the terminal in-band.

Example

final buffer = StringBuffer();
buffer.writeAnsi(AsciiControl.lineFeed);
print(buffer.toString()); // '\n'

backspace → const AsciiControl

Backspace.

Moves the cursor one position to the left, deleting the character there.

Output

Assuming a starting cursor position of █:

Hello█

Writing a backspace:

sink.writeAnsi(AsciiControl.backspace);

Deletes the character at the cursor, moving the cursor back:

Hell█

A backspace at the start of a line has no effect.

Horizontal tab.

Moves the cursor to the next tab stop.

Output

Assuming a starting cursor position of █:

Writing a horizontal tab:

lineFeed → const AsciiControl

Line feed.

Moves the cursor to the next line.

On Unix-like systems, this is the newline character.

Output

Assuming a starting cursor position of █:

Hello█

Writing a line feed:

sink.writeAnsi(AsciiControl.lineFeed);

Moves the cursor to the next line:

Hello
█

On Windows, a line feed is often followed by a carriageReturn to move the cursor to the start of the next line. On POSIX systems, a line feed is often sufficient to move the cursor to the next line.
