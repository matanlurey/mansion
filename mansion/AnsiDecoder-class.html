AnsiDecoder

AnsiDecoder class final Parsing ANSI Text

Converts a string with ANSI escape codes to a list of Sequences.

By default, throws a FormatException if any input is malformed, i.e. contains unrecognized or incomplete escape codes. Set allowInvalid to true to replace malformed input with Unknown sequences instead.

There are two ways to use this decoder:

  1. As a 1-off conversion, AnsiDecoder().convert(input).
  2. As a chunked conversion, AnsiDecoder().startChunkedConversion(sink).

The latter is necessary when decoding a stream of input, as it allows the decoder to buffer incomplete escape codes between chunks.

Example

final decoder = AnsiDecoder();
final decoded = decoder.convert('\x1b[31mHello, World!\x1b[0m');

final decoder = AnsiDecoder();
final sink = decoder.startChunkedConversion(outputSink);

// Would throw a FormatException if this wasn't a chunked conversion.
sink.add('\x1b[');
sink.add('31mHello, World!\x1b[0m');
sink.close();
