Escape

Escape class sealed Introduction

Base type for escape codes, a sequence with a special non-literal meaning.

This type is a marker interface for escape codes to allow for more specific handling of escape codes in a sequence of characters, i.e. by using pattern matching or type checking:

for (final sequence in sequences) {
  switch (sequence) {
    case Print p:
      // Handle literal text.
    case Escape e:
      // Handle escape code.
      // In practice, you might also match specific escape codes.
  }
}
